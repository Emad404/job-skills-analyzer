# Job Skills Analyzer

An AI-powered web application that analyzes any job role and returns a curated list of skills, tools, and certifications demanded by the current job market. Built with Next.js, Tailwind CSS, and the Google Gemini API.

## Features

- **Instant Requirements:** Type a job title and get high-quality, structured requirements generated by Google's Gemini AI.
- **Categorized Breakdown:** Automatically segments results into 'Skills', 'Tools', and 'Certifications'.
- **Lightning Fast Caching:** Caches previous searches in LocalStorage using a cache-first approach to ensure instant responses and reduce API calls.
- **Beautiful UI:** Responsive, modern design with an intelligent Light/Dark theme switcher.
- **Secure Handling:** JSON guardrails ensure data returned by AI is structured correctly, handled natively, and rendered gracefully.

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** React Hooks
- **AI Integration:** [Google Gemini API](https://aistudio.google.com/) (`@google/generative-ai` SDK)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Theming:** `next-themes`

## Getting Started

### 1. Clone the repository

```bash
git clone <your-repository-url>
cd job-skills-analyzer
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure the Environment

Rename the included `.env.local.example` to `.env.local`, or simply create a new `.env.local` file at the root of the project. Retrieve a free Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey) and insert it:

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

*(Note: `.env.local` is ignored by Git, keeping your key secure.)*

### 4. Run the Dev Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) using your browser to see the result.

## Architecture & Logic

- **API Route (app/api/analyze/route.ts):** Secure serverless function that holds the system prompt, handles communication with the Gemini model natively, and applies regex-based JSON guardrails to guarantee accurate formatting.
- **Status Machine (page.tsx):** Smart UI transitioning (idle, loading, success, unrecognized, error) managing graceful fallbacks based on specific API status codes.
- **Client Cache (lib/cache.ts):** Checks browser `localStorage` using versioned keys before pinging the backend API, resulting in near-instant cached searches.
